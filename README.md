## 개시 사이트

## Start

```
npm run start
```

## 개요

파이널 판타지 14 PVP에서 매일 바뀌는 전장 맵을 알려줍니다.

## 기능

- 접속 시 오늘의 전장 맵을 알려줍니다.

- 선택한 날짜의 전장을 알려줍니다.

- 한국어, 영어, 일본어를 지원합니다.
